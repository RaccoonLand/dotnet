# 8. IRequestDispatcher

`IRequestDispatcher` is the host-agnostic entry point for running a request through the command or query
pipeline and receiving a [`PipelineResponse`](7.PipelineResponse.md) envelope.

- **Package:** `RaccoonLand.Core.RequestProcessing.Abstractions`
- **Namespace:** `RaccoonLand.Core.RequestProcessing.Abstractions.Dispatch`

## Surface

```csharp
public interface IRequestDispatcher
{
    Task<PipelineResponse?> DispatchAsync(
        IRequestBase request,
        IServiceProvider requestServices,
        CancellationToken cancellationToken = default);
}
```

| Parameter | Description |
| --- | --- |
| `request` | The command or query to process. Must implement [`IRequestBase`](1.Cqrs.md). |
| `requestServices` | The **per-request scope** — endpoints and scoped middleware are resolved from here. |
| `cancellationToken` | A token tied to the request lifetime, forwarded to the endpoint. |

The return value is the [`PipelineResponse`](7.PipelineResponse.md) envelope produced by the pipeline (or
`null` if no terminal/middleware set one).

## Host independence

The interface has no dependency on ASP.NET Core. A host adapter is responsible for:

- **API** — open a request scope (`HttpContext.RequestServices`) and forward `HttpContext.RequestAborted`.
- **Worker** — create a scope from the root provider and forward the worker's stopping token.
- **Tests** — pass a test scope and `CancellationToken.None`.

## Default implementation

The concrete [`RequestDispatcher`](../../RequestProcessing/docs/5.RequestDispatcher.md) lives in
`RaccoonLand.Core.RequestProcessing`. It looks up the request's pre-registered
[`RequestKind`](3.PipelineContext.md) from the [`EndpointInvokerRegistry`](../../RequestProcessing/docs/3.EndpointInvokerRegistry.md)
(so no run-time reflection is needed to classify command vs. query) and runs the matching compiled pipeline.

Register everything with [`AddRaccoonLandRequestProcessing`](../../RequestProcessing/docs/6.Registration.md).

See also: [PipelineContext](3.PipelineContext.md), [CQRS contracts](1.Cqrs.md).
