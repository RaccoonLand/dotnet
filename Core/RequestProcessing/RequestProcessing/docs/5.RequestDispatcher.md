# 5. RequestDispatcher

`RequestDispatcher` is the default implementation of
[`IRequestDispatcher`](../../RequestProcessing.Abstractions/docs/8.IRequestDispatcher.md). It looks up the
request's pre-registered [`RequestKind`](../../RequestProcessing.Abstractions/docs/3.PipelineContext.md),
builds a [`PipelineContext`](../../RequestProcessing.Abstractions/docs/3.PipelineContext.md), and runs it
through the matching [compiled pipeline](4.CompiledPipelines.md).

- **Package:** `RaccoonLand.Core.RequestProcessing`
- **Namespace:** `RaccoonLand.Core.RequestProcessing.Dispatch`

## Surface

The public contract is defined in the abstractions package:

```csharp
// RaccoonLand.Core.RequestProcessing.Abstractions.Dispatch
public interface IRequestDispatcher
{
    Task<PipelineResponse?> DispatchAsync(
        IRequestBase request,
        IServiceProvider requestServices,
        CancellationToken cancellationToken = default);
}
```

## Behaviour

```csharp
var kind     = _registry.ResolveKind(request.GetType());
var pipeline = kind == RequestKind.Query ? _pipelines.Query : _pipelines.Command;
if (pipeline is null) throw new InvalidOperationException(/* not built */);

var context = new PipelineContext(request, kind, requestServices, cancellationToken);
await pipeline(context);
return context.Response;
```

Classification happens **once at startup** when endpoints are registered in the
[`EndpointInvokerRegistry`](3.EndpointInvokerRegistry.md) (via
[`AddRaccoonLandRequestProcessing`](6.Registration.md)). The dispatcher only reads the stored
`RequestKind` — no run-time reflection to detect `IQuery<>`.

## Dependencies

| Dependency | Role |
| --- | --- |
| `CompiledPipelines` | The two built `PipelineDelegate`s (command and query). |
| `EndpointInvokerRegistry` | Supplies `ResolveKind` for the request type. |

## Lifecycle and threading

`RequestDispatcher` is stateless beyond its dependencies, so it is safe to register as a singleton and call
concurrently. All per-request state lives on the `PipelineContext` created for each call.

See also: [IRequestDispatcher (contract)](../../RequestProcessing.Abstractions/docs/8.IRequestDispatcher.md),
[Registration](6.Registration.md).
