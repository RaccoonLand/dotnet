# 5. Registration

- **Package:** `RaccoonLand.Core.Hosting.AspNetCore`

## Project layout (`src/`)

Each feature lives in its own folder with all related types and extensions:

| Folder | Namespace | Contents |
| --- | --- | --- |
| `Hosting/` | `.Hosting` | `AddRaccoonLandAspNetCore` (umbrella registration) |
| `Controllers/` | `.Controllers` | `RaccoonLandController` |
| `PipelineResponseMapping/` | `.PipelineResponseMapping` | `IPipelineResponseMapper`, `DefaultPipelineResponseMapper` |
| `HttpExecutionContext/` | `.HttpExecutionContext` | Context types, middleware, `Add*` / `Use*` extensions |
| `HttpExceptionHandling/` | `.HttpExceptionHandling` | `HttpExceptionHandlingOptions`, `HttpExceptionHandlingMiddleware`, `Add*` / `Use*` extensions |

## AddRaccoonLandAspNetCore

```csharp
namespace RaccoonLand.Core.Hosting.AspNetCore.Hosting;

public static IServiceCollection AddRaccoonLandAspNetCore(
    this IServiceCollection services,
    IConfiguration? configuration = null,
    Action<HttpExecutionContextOptions>? configureExecutionContext = null);
```

| Registers | When |
| --- | --- |
| `IPipelineResponseMapper` → `DefaultPipelineResponseMapper` | Always |
| `ICurrentExecutionContext` → `HttpExecutionContext` | When `configuration` or `configureExecutionContext` is supplied |

## Full startup example

```csharp
using RaccoonLand.Core.Hosting.AspNetCore.HttpExceptionHandling;
using RaccoonLand.Core.Hosting.AspNetCore.Hosting;
using RaccoonLand.Core.Hosting.AspNetCore.HttpExecutionContext;
using RaccoonLand.Core.RequestProcessing.DependencyInjection;
using RaccoonLand.Modules.Middlewares.FluentValidationMiddleware;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddControllers();
builder.Services.AddRaccoonLandAspNetCore(builder.Configuration);
builder.Services.AddRaccoonLandHttpExceptionHandling();

builder.Services.AddRaccoonLandRequestProcessing(
    configureCommandPipeline: pipeline =>
    {
        pipeline.UseMiddleware<FluentValidationMiddleware>();
    },
    configureQueryPipeline: pipeline =>
    {
        pipeline.UseMiddleware<FluentValidationMiddleware>();
    },
    typeof(Program).Assembly);

var app = builder.Build();

app.UseHttpsRedirection();
app.UseRaccoonLandHttpExceptionHandling();
app.UseAuthentication();
app.UseRaccoonLandHttpExecutionContext();
app.UseAuthorization();
app.MapControllers();
app.Run();
```
