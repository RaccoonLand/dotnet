# 3. HttpExecutionContext

HTTP-scoped implementation of
[`ICurrentExecutionContext`](../../ExecutionContext/docs/1.ICurrentExecutionContext.md). Values are resolved from
JWT/OIDC claims and request headers according to [`HttpExecutionContextOptions`](3.HttpExecutionContext.md#options),
then exposed to audit, outbox, and handlers for the lifetime of the request.

- **Package:** `RaccoonLand.Core.Hosting.AspNetCore`
- **Namespace:** `RaccoonLand.Core.Hosting.AspNetCore.HttpExecutionContext`
- **Folder:** `src/HttpExecutionContext/` (all types and extensions for this feature)

## Components

| Type | Role |
| --- | --- |
| `HttpExecutionContextOptions` | Claim/header mapping (appsettings + `Action` configure). |
| `HttpExecutionContext` | Scoped `ICurrentExecutionContext` implementation. |
| `HttpExecutionContextPopulator` | Internal claim/header resolution. |
| `HttpExecutionContextMiddleware` | Populates the context once per HTTP request. |
| `AddRaccoonLandHttpExecutionContext` | DI registration. |
| `UseRaccoonLandHttpExecutionContext` | HTTP pipeline middleware (after `UseAuthentication`). |

## Options

```csharp
public sealed class HttpExecutionContextOptions
{
    public const string SectionName = "HttpExecutionContext";

    public string UserIdClaim { get; set; } = "sub";
    public string? TenantIdClaim { get; set; }
    public string? TenantIdHeader { get; set; }
    public string CorrelationIdHeader { get; set; } = "X-Correlation-Id";
    public bool GenerateCorrelationIdWhenMissing { get; set; } = true;
    public bool EchoCorrelationIdInResponse { get; set; } = true;
}
```

### Resolution order

| Field | Sources (in order) |
| --- | --- |
| `UserId` | Claim `UserIdClaim` |
| `TenantId` | Claim `TenantIdClaim`, then header `TenantIdHeader` |
| `CorrelationId` | Header `CorrelationIdHeader`, then `Activity.Current.TraceId`, then generated GUID v7 when enabled |

`IsAvailable` is `true` when at least one of `UserId`, `TenantId`, or `CorrelationId` is non-null.

## Registration

### From appsettings + optional Action

```json
{
  "HttpExecutionContext": {
    "UserIdClaim": "sub",
    "TenantIdClaim": "tenant_id",
    "TenantIdHeader": "X-Tenant-Id",
    "CorrelationIdHeader": "X-Correlation-Id"
  }
}
```

```csharp
builder.Services.AddRaccoonLandHttpExecutionContext(
    builder.Configuration,
    configureOptions: o => o.TenantIdHeader = "X-Tenant-Id");

// or via AddRaccoonLandAspNetCore when configuration is passed:
builder.Services.AddRaccoonLandAspNetCore(builder.Configuration);
```

### Code-only

```csharp
builder.Services.AddRaccoonLandHttpExecutionContext(o =>
{
    o.UserIdClaim = "oid";
    o.TenantIdHeader = "X-Tenant-Id";
});
```

### HTTP pipeline

```csharp
app.UseAuthentication();
app.UseRaccoonLandHttpExecutionContext();
app.UseAuthorization();
```
