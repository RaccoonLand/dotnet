# 1. RaccoonLandBackgroundService

`RaccoonLandBackgroundService` is the generic base class for background workers. It extends
`BackgroundService` and exposes `DispatchAsync` helpers that send a CQRS request through
[`WorkerRequestDispatcher`](2.WorkerRequestDispatcher.md) inside a **fresh DI scope per call**.

- **Package:** `RaccoonLand.Core.Hosting.Worker`
- **Namespace:** `RaccoonLand.Core.Hosting.Worker`

## Surface

```csharp
public abstract class RaccoonLandBackgroundService : BackgroundService
{
    protected Task<WorkerDispatchResult> DispatchAsync(
        IRequest request,
        WorkerExecutionMetadata? metadata = null,
        CancellationToken cancellationToken = default);

    protected Task<WorkerDispatchResult> DispatchAsync<TResponse>(
        IRequest<TResponse> request,
        WorkerExecutionMetadata? metadata = null,
        CancellationToken cancellationToken = default);
}
```

## Responsibilities

| Layer | Responsibility |
| --- | --- |
| **Worker** | Poll/consume work, build `IRequest`, call `DispatchAsync`, decide retry/ack from `WorkerDispatchResult`. |
| **`WorkerRequestDispatcher`** | Open scope, configure execution context, call `IRequestDispatcher`. |
| **`IEndpoint` handler** | Business logic. |
| **Pipeline middleware** | Validation, exception handling, … (same as API). |
| **`IPipelineResponseHandler`** | Interpret `PipelineResponse` → `IsSuccess` (envelope preserved). |

## Example

```csharp
public sealed class ProcessPendingOrdersWorker(WorkerRequestDispatcher dispatcher)
    : RaccoonLandBackgroundService(dispatcher)
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            var orderId = await DequeueNextOrderIdAsync(stoppingToken);

            var result = await DispatchAsync(
                new ProcessOrderCommand(orderId),
                new WorkerExecutionMetadata { CorrelationId = orderId.ToString() },
                stoppingToken);

            if (!result.IsSuccess)
            {
                await HandleFailureAsync(result.Response, stoppingToken);
                continue;
            }

            await AckAsync(orderId, stoppingToken);
        }
    }
}
```

Register the worker as a hosted service:

```csharp
services.AddHostedService<ProcessPendingOrdersWorker>();
```
