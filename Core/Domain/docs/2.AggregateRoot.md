# 2. AggregateRoot

`AggregateRoot<TKey>` extends `Entity<TKey>` and is the consistency boundary of the domain. It also implements the non-generic `IAggregateRoot`, which lets infrastructure discover aggregates without knowing `TKey`.

- Namespace: `RaccoonLand.Core.Domain.Abstractions`
- Package: `RaccoonLand.Core.Domain`

## Responsibilities

1. **Domain events** — collects `DomainEvent` instances to be dispatched/persisted via the Outbox pattern when EF Core saves the aggregate.
2. **Service events** — collects `ServiceEvent` (cross-service integration events), also persisted via the Outbox.
3. **Optimistic concurrency** — owns a `ConcurrencyToken` (GUID) regenerated on each save to detect concurrent modifications.

## Surface

| Member | Description |
|--------|-------------|
| `ConcurrencyToken` | Sequential GUID; configured as an EF Core concurrency token. |
| `DomainEvents` | Read-only collection of raised domain events. |
| `ServiceEvents` | Read-only collection of raised service events. |
| `RaiseDomainEvent(...)` | `protected` — raise a domain event from within the aggregate. |
| `RaiseServiceEvent(...)` | `protected` — raise a service event from within the aggregate. |
| `ClearDomainEvents()` / `ClearServiceEvents()` | Cleared by infrastructure after the events are persisted. |
| `RegenerateConcurrencyToken()` | Invoked by the save interceptor. |

## IAggregateRoot

The non-generic contract exposes `BusinessKey`, `ConcurrencyToken`, the event collections, the clear methods, and `RegenerateConcurrencyToken()`. Infrastructure (interceptors, outbox writers) depends on this interface rather than the generic base.

## Example

```csharp
public sealed class Order : AggregateRoot<Guid>
{
    public Order(Guid id) : base(id) { }

    public void Submit()
    {
        // ... state change ...
        RaiseDomainEvent(new OrderSubmitted(BusinessKey));
    }
}
```

## Notes

- `ConcurrencyToken` must be included in every query (read model) output so updates can round-trip it.
- Events are append-only within a unit of work; infrastructure clears them after writing to the outbox.
