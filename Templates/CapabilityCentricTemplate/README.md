# Templates

RaccoonLand .NET starter templates for capability-centric applications.

| Template | Package | Short name | Description |
|----------|---------|------------|-------------|
| [CapabilityCentricTemplate](CapabilityCentricTemplate/docs/1.Overview.md) | `RaccoonLand.Templates.CapabilityCentric` | `raccoonland-capability` | Minimal People capability with Create, GetById, and Search endpoints |

## Install

```bash
dotnet new install RaccoonLand.Templates.CapabilityCentric
```

## Create

```bash
dotnet new raccoonland-capability -n MyCompany.People -o ./MyCompany.People
```
