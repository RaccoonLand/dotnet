# 2. Installation

## Prerequisites

- .NET SDK `10.0.301` or newer
- SQL Server for persistence
- Access to RaccoonLand packages from the configured NuGet feed

The generated solution includes a local `nuget.config` that maps `RaccoonLand.*` packages to:

```text
https://localhost:5001/v3/index.json
```

## Install the template package

From a machine that can reach the NuGet feed:

```bash
dotnet new install RaccoonLand.Templates.CapabilityCentric
```

Verify installation:

```bash
dotnet new list raccoonland
```

Expected output includes:

```text
RaccoonLand Capability-Centric Template    raccoonland-capability
```

## Update the template

```bash
dotnet new uninstall RaccoonLand.Templates.CapabilityCentric
dotnet new install RaccoonLand.Templates.CapabilityCentric
```

Or install a specific version:

```bash
dotnet new install RaccoonLand.Templates.CapabilityCentric@1.0.0
```

## Uninstall

```bash
dotnet new uninstall RaccoonLand.Templates.CapabilityCentric
```
