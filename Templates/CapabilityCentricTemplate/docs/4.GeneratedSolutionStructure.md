```text
{ProjectName}/
├── global.json
├── nuget.config
├── Directory.Build.props
├── Directory.Packages.props
├── {ProjectName}.slnx
└── src/
    ├── {ProjectName}/
    └── {ProjectName}.Hosting.API/
```

## Application project

```text
src/{ProjectName}/
├── People/
│   ├── Domain/
│   ├── Endpoints/
│   ├── Persistence/
│   └── Shared/
└── Shared/
    ├── Localizations/
    └── Persistence/

```

## Host project

```text
src/{ProjectName}.Hosting.API/
├── Controllers/
│   └── People/

├── Program.cs

```

## DbContext placement

| DbContext | File | Responsibility |
|-----------|------|----------------|
| `{ProjectName}CommandDbContext` or custom name | `Shared/Persistence/Commands/{Name}.cs` | Writes, future migrations, outbox interceptors |
| `{ProjectName}QueryDbContext` or custom name | `Shared/Persistence/Queries/{Name}.cs` | Read-only queries |

## People capability

| Layer | Key types |
|-------|-----------|
| Domain | `Person`, `FirstName`, `LastName` |
| Commands | `CreatePersonCommand`, `CreatePersonEndpoint` |
| Queries | `GetPersonByIdQuery`, `SearchPeopleQuery` |
| Persistence | `PersonConfiguration`, query model `Person` |

## HTTP routes

| Route | Request |
|-------|---------|
| `POST /api/people` | `CreatePersonCommand` |
| `GET /api/people/{id}` | `GetPersonByIdQuery` |
| `GET /api/people/Search` | `SearchPeopleQuery` |

## First run checklist

1. Update SQL connection strings in `appsettings.json`.
2. Create the initial migration:

```bash
dotnet ef migrations add Init --project src/{ProjectName} --startup-project src/{ProjectName}.Hosting.API --context {CommandDbContextName} --output-dir Shared/Persistence/Commands/Migrations
```

3. Apply the migration:

```bash
dotnet ef database update --project src/{ProjectName} --startup-project src/{ProjectName}.Hosting.API
```

4. Run the host:

```bash
dotnet run --project src/{ProjectName}.Hosting.API
```

5. Open Scalar:

```text
/scalar/v1
```
