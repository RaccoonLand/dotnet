# Templates

RaccoonLand .NET starter templates.

| Template | Package | Short name | Description |
|----------|---------|------------|-------------|
| [CapabilityCentricTemplate](../CapabilityCentricTemplate/docs/1.Overview.md) | `RaccoonLand.Templates.CapabilityCentric` | `raccoonland-capability` | Minimal People capability with Create, GetById, and Search endpoints |
| [CleanArchitectureTemplate](docs/1.Overview.md) | `RaccoonLand.Templates.CleanArchitecture` | `raccoonland-clean-architecture` | Layered clean architecture with Domain, Application, Infrastructure, and Hosting projects |

## Install

```bash
dotnet new install RaccoonLand.Templates.CleanArchitecture
```

## Create

```bash
dotnet new raccoonland-clean-architecture -n MyCompany.People -o ./MyCompany.People
```
