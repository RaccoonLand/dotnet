# 3. Template Parameters

## Built-in project name

The standard `dotnet new` name parameter controls the generated project name and namespace root.

```bash
dotnet new raccoonland-clean-architecture -n MyCompany.People -o ./MyCompany.People
```

This replaces the template token `CleanArchitectureTemplate` in:

- folder names
- project file names
- namespaces
- configuration keys that include the project namespace

Extension methods use the `TemplateApp` token and become names such as `AddMyCompanyPeopleApi` and `AddMyCompanyPeopleCommandsPersistence`.

## Command DbContext

Parameter:

```bash
--CommandDbContextName
```

Example:

```bash
dotnet new raccoonland-clean-architecture `
  -n MyCompany.People `
  -o ./MyCompany.People `
  --CommandDbContextName MyCompanyPeopleCommandDbContext
```

Behavior:

- Replaces token `TemplateCommandDbContext`
- Renames `TemplateCommandDbContext.cs`
- Becomes the DbContext name you will use when creating the first EF Core migration

Default when omitted:

```text
{SafeProjectName}CommandDbContext
```

`{SafeProjectName}` is derived from the project name by removing characters that are invalid in C# identifiers (for example `Acme.HumanResources` becomes `AcmeHumanResources`).

## Query DbContext

Parameter:

```bash
--QueryDbContextName
```

Example:

```bash
dotnet new raccoonland-clean-architecture `
  -n MyCompany.People `
  -o ./MyCompany.People `
  --QueryDbContextName MyCompanyPeopleQueryDbContext
```

Behavior:

- Replaces token `TemplateQueryDbContext`
- Renames `TemplateQueryDbContext.cs`
- Updates DI registration

Default when omitted:

```text
{SafeProjectName}QueryDbContext
```

## Full example

```bash
dotnet new raccoonland-clean-architecture `
  -n Acme.HumanResources `
  -o ./Acme.HumanResources `
  --CommandDbContextName AcmeHumanResourcesCommandDbContext `
  --QueryDbContextName AcmeHumanResourcesQueryDbContext
```

Generated DbContext classes:

- `AcmeHumanResourcesCommandDbContext`
- `AcmeHumanResourcesQueryDbContext`

## Dry run

Inspect the generated output without writing files:

```bash
dotnet new raccoonland-clean-architecture `
  -n Preview.App `
  --CommandDbContextName PreviewCommandDbContext `
  --QueryDbContextName PreviewQueryDbContext `
  --dry-run
```
