# 2. Enums

## ServiceLifetime

| Value | Meaning |
|-------|---------|
| `Singleton` | One instance for the application root container |
| `Scoped` | One instance per request scope |
| `Transient` | A new instance on every resolve |

## ServiceRegistration

| Value | Scrutor mapping | Meaning |
|-------|-----------------|---------|
| `Self` | `AsSelf()` | Register only the concrete type |
| `MatchingInterface` | `AsMatchingInterface()` | Register against non-generic `I{ClassName}` (same namespace preferred) |
| `ImplementedInterfaces` | `AsImplementedInterfaces()` | Register against every implemented interface |
| `SelfAndImplementedInterfaces` | `AsSelfWithInterfaces()` | Register concrete + interfaces together; shared instance for Scoped/Singleton only |
