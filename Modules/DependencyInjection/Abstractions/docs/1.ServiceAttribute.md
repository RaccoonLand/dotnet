# 1. ServiceAttribute

`ServiceAttribute` marks application services for automatic registration through
`AddRaccoonLandDependencyInjection`.

- **Package:** `RaccoonLand.Modules.DependencyInjection.Abstractions`
- **Namespace:** `RaccoonLand.Modules.DependencyInjection.Abstractions`

## Example

```csharp
[Service(
    Lifetime = ServiceLifetime.Scoped,
    Registration = ServiceRegistration.ImplementedInterfaces)]
public sealed class PersonService : IPersonService
{
}
```

## Properties

| Property | Default | Description |
|----------|---------|-------------|
| `Lifetime` | `Scoped` | Container lifetime |
| `Registration` | `ImplementedInterfaces` | How the concrete type is exposed in DI |

## Notes

- Apply the attribute to **concrete** classes only. Abstract classes are ignored.
- Only one attribute per class is supported.
- Bring `RaccoonLand.Modules.DependencyInjection.MicrosoftDI` and call
  `AddRaccoonLandDependencyInjection` during startup.
