# 1. Registration

`AddRaccoonLandDependencyInjection` scans assemblies for classes decorated with
[`ServiceAttribute`](../../Abstractions/docs/1.ServiceAttribute.md) and registers them with Scrutor.

- **Package:** `RaccoonLand.Modules.DependencyInjection.MicrosoftDI`
- **Namespace:** `Microsoft.Extensions.DependencyInjection`

## Surface

```csharp
public static IServiceCollection AddRaccoonLandDependencyInjection(
    this IServiceCollection services,
    params Assembly[] assemblies);
```

When `assemblies` is omitted, the **calling assembly** is scanned.

## Example

```csharp
using System.Reflection;

builder.Services.AddRaccoonLandDependencyInjection(typeof(Program).Assembly);
```

Or scan multiple assemblies:

```csharp
builder.Services.AddRaccoonLandDependencyInjection(
    typeof(Program).Assembly,
    typeof(SomeOtherMarker).Assembly);
```

## Prerequisites

Reference both packages:

- `RaccoonLand.Modules.DependencyInjection.Abstractions`
- `RaccoonLand.Modules.DependencyInjection.MicrosoftDI`

See also: [2. Scanning Rules](2.ScanningRules.md).
