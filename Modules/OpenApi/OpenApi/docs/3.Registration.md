# 3. Registration

Two extension methods wire the OpenAPI core into an ASP.NET Core app.

- **Package:** `RaccoonLand.Modules.OpenApi`
- **Namespace:** `RaccoonLand.Modules.OpenApi`

## Surface

```csharp
// DI — finalizes OpenApiDocumentOptions and registers the document + transformers
public static IServiceCollection AddRaccoonLandOpenApi(
    this IServiceCollection services,
    IConfiguration configuration,
    string sectionName = OpenApiDocumentOptions.SectionName,
    Action<OpenApiDocumentOptions>? configureOptions = null);

// Endpoints — serves the OpenAPI JSON at OpenApiDocumentOptions.RoutePattern
public static IEndpointRouteBuilder MapRaccoonLandOpenApi(this IEndpointRouteBuilder endpoints);
```

| Member | Description |
| --- | --- |
| `AddRaccoonLandOpenApi` | Binds options once, applies `configureOptions` once, validates, registers `IOptions` snapshot, and (when `Enabled`) calls `AddOpenApi`. |
| `MapRaccoonLandOpenApi` | Maps the document endpoint from the same snapshot when `Enabled`; otherwise a no-op. |

## Binding precedence

1. Configuration is bound from the root section.
2. `configureOptions` is applied once as a code override.
3. Options are validated (`DocumentName`, `RoutePattern` with `{documentName}`, `Security` / `SchemeName`).
4. The same instance is registered as `IOptions<OpenApiDocumentOptions>` and used for document registration.

This avoids running `configureOptions` twice and keeps `Enabled` / route / document name consistent between
registration and `Map*` / UI providers.

## Validation (fail-fast)

Startup fails with `InvalidOperationException` when:

- `DocumentName` or `RoutePattern` is null, empty, or whitespace.
- `RoutePattern` does not contain `{documentName}`.
- `Security` is null, or JWT Bearer is enabled with an empty `SchemeName`.

## Example

```csharp
using RaccoonLand.Modules.OpenApi;

var builder = WebApplication.CreateBuilder(args);
builder.Services.AddRaccoonLandOpenApi(builder.Configuration);

var app = builder.Build();
app.MapRaccoonLandOpenApi();   // GET /openapi/v1.json
app.Run();
```

Add a UI provider for a browsable reference:

```csharp
app.MapRaccoonLandScalar();    // RaccoonLand.Modules.OpenApi.Scalar
app.UseRaccoonLandSwaggerUI(); // RaccoonLand.Modules.OpenApi.Swagger
```
