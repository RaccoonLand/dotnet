# 2. BearerSecuritySchemeDocumentTransformer

`BearerSecuritySchemeDocumentTransformer` is an `IOpenApiDocumentTransformer` that adds a JWT Bearer security
scheme to the generated OpenAPI document (and, when configured, a global security requirement) so a UI shows
an Authorize dialog.

- **Package:** `RaccoonLand.Modules.OpenApi`
- **Namespace:** `RaccoonLand.Modules.OpenApi`
- **Visibility:** `internal` — registered by `AddRaccoonLandOpenApi` when `Security.EnableJwtBearer` is `true`.

## Behaviour

```csharp
internal sealed class BearerSecuritySchemeDocumentTransformer(OpenApiSecurityOptions security)
    : IOpenApiDocumentTransformer
{
    public Task TransformAsync(OpenApiDocument document, OpenApiDocumentTransformerContext context, CancellationToken ct)
    {
        // SchemeName validated non-empty; existing component with the same name fails fast.
        var scheme = new OpenApiSecurityScheme { /* Http bearer … */ };

        document.Components ??= new OpenApiComponents();
        if (document.Components.SecuritySchemes?.ContainsKey(security.SchemeName) == true)
            throw new InvalidOperationException(/* duplicate component */);

        document.AddComponent(security.SchemeName, scheme);

        if (security.ApplyGlobally)
        {
            document.Security ??= [];
            document.Security.Add(new OpenApiSecurityRequirement
            {
                [new OpenApiSecuritySchemeReference(security.SchemeName, document)] = [],
            });
        }

        return Task.CompletedTask;
    }
}
```

## .NET 10 OpenAPI reference API

.NET 10 changed the OpenAPI object model: a security requirement references a scheme through
`OpenApiSecuritySchemeReference(name, document)` after the scheme is added to the document via
`AddComponent`. Assigning a `Reference` directly on the scheme (the .NET 8/9 pattern) no longer compiles.

## Notes

- Registered as a transformer **instance** (constructed with the finalized `OpenApiSecurityOptions` snapshot),
  not a DI-activated transformer.
- Duplicate `SchemeName` components throw; they are not overwritten silently.
- This adds metadata only; it does not enforce authentication. See
  [OpenApiSecurityOptions](../../Abstractions/docs/2.OpenApiSecurityOptions.md).
