# 2. Registration

- **Package:** `RaccoonLand.Modules.OpenApi.Swagger`
- **Namespace:** `RaccoonLand.Modules.OpenApi.Swagger`

## Surface

```csharp
public static IApplicationBuilder UseRaccoonLandSwaggerUI(
    this IApplicationBuilder app,
    Action<SwaggerUIOptions>? configure = null);
```

| Member | Description |
| --- | --- |
| `UseRaccoonLandSwaggerUI` | Mounts Swagger UI against the configured OpenAPI document route when `OpenApiDocumentOptions.Enabled` is `true`; otherwise a no-op. `configure` exposes the underlying `SwaggerUIOptions` (route prefix, theme, …). |

## Example

```csharp
using RaccoonLand.Modules.OpenApi;        // core
using RaccoonLand.Modules.OpenApi.Swagger; // this provider

var builder = WebApplication.CreateBuilder(args);
builder.Services.AddRaccoonLandOpenApi(builder.Configuration);

var app = builder.Build();
app.MapRaccoonLandOpenApi();    // GET /openapi/v1.json
app.UseRaccoonLandSwaggerUI();  // Swagger UI at /swagger
app.Run();
```

Customize the UI:

```csharp
app.UseRaccoonLandSwaggerUI(ui =>
{
    ui.RoutePrefix = "docs";
    ui.DocumentTitle = "My API";
});
```
