# 2. Registration

- **Package:** `RaccoonLand.Modules.OpenApi.Scalar`
- **Namespace:** `RaccoonLand.Modules.OpenApi.Scalar`

## Surface

```csharp
public static IEndpointRouteBuilder MapRaccoonLandScalar(
    this IEndpointRouteBuilder endpoints,
    Action<ScalarOptions>? configure = null);
```

| Member | Description |
| --- | --- |
| `MapRaccoonLandScalar` | Maps the Scalar reference against the configured OpenAPI document route when `OpenApiDocumentOptions.Enabled` is `true`; otherwise a no-op. `configure` exposes the underlying `ScalarOptions` (theme, layout, auth, …). |

## Example

```csharp
using RaccoonLand.Modules.OpenApi;       // core
using RaccoonLand.Modules.OpenApi.Scalar; // this provider

var builder = WebApplication.CreateBuilder(args);
builder.Services.AddRaccoonLandOpenApi(builder.Configuration);

var app = builder.Build();
app.MapRaccoonLandOpenApi();  // GET /openapi/v1.json
app.MapRaccoonLandScalar();   // Scalar UI at /scalar/v1
app.Run();
```

Customize the UI:

```csharp
app.MapRaccoonLandScalar(scalar =>
{
    scalar.WithTheme(ScalarTheme.Mars)
          .WithSidebar(true);
});
```
