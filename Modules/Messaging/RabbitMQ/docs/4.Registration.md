## Full wiring with outbox relay

```csharp
builder.Services.AddRaccoonLandOutboxEventStore(builder.Configuration, configureSql: sql =>
{
    sql.ConnectionString = builder.Configuration.GetConnectionString("Commands")!;
});

builder.Services.AddRaccoonLandDomainEventHandler<OrderSubmitted, OrderSubmittedHandler>();

builder.Services.AddRaccoonLandRabbitMqServiceEvents(builder.Configuration);

builder.Services.AddRaccoonLandOutboxRelay(builder.Configuration, configure: relay =>
{
    relay.ProcessDomainEvents = true;
    relay.ProcessServiceEvents = true;
});
```

`appsettings.json`:

```json
{
  "OutboxRelay": {
    "ProcessDomainEvents": true,
    "ProcessServiceEvents": true
  },
  "RabbitMqServiceEvents": {
    "Uri": "amqp://guest:guest@localhost:5672/",
    "ExchangeName": "raccoonland.service-events",
    "ExchangeType": "topic",
    "RoutingKeyFormat": "{EventType}"
  }
}
```

Without `ProcessServiceEvents = true`, Service rows stay pending in SQL even if the publisher is registered.
