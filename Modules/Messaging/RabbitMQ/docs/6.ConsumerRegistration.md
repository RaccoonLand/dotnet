# 6. Consumer registration

```csharp
public static IServiceCollection AddRaccoonLandRabbitMqServiceEventConsumer(
    this IServiceCollection services,
    IConfiguration configuration,
    string sectionName = RabbitMqServiceEventConsumerOptions.SectionName,
    Action<RabbitMqServiceEventConsumerOptions>? configure = null);

public static IServiceCollection AddRaccoonLandServiceEventHandler<TEvent, THandler>(
    this IServiceCollection services,
    ServiceLifetime lifetime = ServiceLifetime.Scoped)
    where TEvent : ServiceEvent
    where THandler : class, IServiceEventHandler<TEvent>;
```

## Full consuming host example

```csharp
builder.Services.AddRaccoonLandInboxStore(builder.Configuration, configureSql: sql =>
{
    sql.ConnectionString = builder.Configuration.GetConnectionString("Commands")!;
});

builder.Services.AddRaccoonLandServiceEventHandler<OrderShippedIntegrationEvent, OrderShippedHandler>();
builder.Services.AddRaccoonLandRabbitMqServiceEventConsumer(builder.Configuration);
```

```json
{
  "InboxStore": {
    "Schema": "dbo",
    "Table": "InboxEvent",
    "ConnectionString": "..."
  },
  "RabbitMqServiceEventConsumer": {
    "Uri": "amqp://guest:guest@localhost:5672/",
    "ExchangeName": "raccoonland.service-events",
    "ExchangeType": "topic",
    "QueueName": "ordering-service.service-events",
    "BindingKeys": [ "order.#" ],
    "PrefetchCount": 10,
    "InboxClaimLease": "00:05:00",
    "ClaimHeldByOtherRequeueDelay": "00:00:02",
    "MaxDeliveryAttempts": 5,
    "EnableDeadLetterTopology": true
  }
}
```

Lease duration is **`RabbitMqServiceEventConsumer:InboxClaimLease`**, not a property on `InboxStore`.

| Registers | Lifetime |
| --- | --- |
| `RabbitMqServiceEventConsumerOptions` | Options |
| `ServiceEventHandlerRegistry` | Singleton |
| `IServiceEventDispatcher` → `ServiceEventDispatcher` | Singleton |
| `RabbitMqServiceEventConsumerBackgroundService` | Hosted service |
| `IServiceEventHandler<T>` / `THandler` | Scoped by default |

Requires `IInboxStore`. Does not require the publisher registration on the same host (consuming services usually only consume).

See also: [Consumer behaviour](5.Consumer.md), [Inbox SQL](../SqlServer/docs/4.Inbox.md).
