# 0. Overview — Messaging (outbox consumption)

RaccoonLand already **raises** domain/service events on aggregates and **writes** them transactionally to
`OutboxEvent` via `OutboxSaveChangesInterceptor`. This Messaging area covers the **next stage**:
asynchronous **consumption** outside the originating HTTP/command request.

## Design decisions

| Decision | Choice |
| --- | --- |
| Domain vs Service | Same outbox table; routed by `Category` (`Domain` / `Service`). |
| When Domain handlers run | **Worker**, not the originating request (latency / isolation). |
| Domain transport | In-process dispatcher in the same service (no broker required). |
| Service transport | `IServiceEventPublisher` → RabbitMQ adapter publishes to a topic exchange. |
| Delivery | At-least-once; consumers must be idempotent on `EventId`. |

## Packages

| Package | Role |
| --- | --- |
| `RaccoonLand.Modules.Messaging.Abstractions` | Contracts: store, dispatcher, publisher/consumer, inbox, options. |
| `RaccoonLand.Modules.Messaging.SqlServer` | `IOutboxEventStore` + `IInboxStore` for SQL Server. |
| `RaccoonLand.Modules.Messaging.OutboxRelay` | Domain dispatcher + outbox relay worker. |
| `RaccoonLand.Modules.Messaging.RabbitMQ` | Service-event publisher **and** inbox-gated consumer. |

## End-to-end flow

```text
Producing service
  Command → OutboxEvent → Relay
    Domain → IDomainEventHandler (in-process)
    Service → RabbitMQ exchange

Consuming service
  Queue bind → Consumer worker
    Inbox claim (EventId)
    → IServiceEventHandler<T>
    → Mark processed → ACK
```

## Schema additions

**Outbox (producer):**

```sql
ALTER TABLE dbo.OutboxEvent ADD ProcessedOnUtc datetimeoffset NULL;
ALTER TABLE dbo.OutboxEvent ADD ClaimedOnUtc   datetimeoffset NULL;
```

**Inbox (consumer):**

```sql
CREATE TABLE dbo.InboxEvent
(
    EventId        uniqueidentifier NOT NULL CONSTRAINT PK_InboxEvent PRIMARY KEY,
    EventType      nvarchar(256)    NOT NULL,
    ClaimedOnUtc   datetimeoffset   NULL,
    ProcessedOnUtc datetimeoffset   NULL,
    ReceivedOnUtc  datetimeoffset   NOT NULL
);
```

See [SqlServer outbox schema](../SqlServer/docs/2.Schema.md) and [Inbox](../SqlServer/docs/4.Inbox.md).

## Suggested producer wiring

```csharp
builder.Services.AddRaccoonLandOutboxEventStore(...);
builder.Services.AddRaccoonLandDomainEventHandler<OrderSubmitted, OrderSubmittedHandler>();
builder.Services.AddRaccoonLandRabbitMqServiceEvents(builder.Configuration);
builder.Services.AddRaccoonLandOutboxRelay(builder.Configuration); // ProcessServiceEvents = true
```

## Suggested consumer wiring

```csharp
builder.Services.AddRaccoonLandInboxStore(...);
builder.Services.AddRaccoonLandServiceEventHandler<OrderShippedIntegrationEvent, OrderShippedHandler>();
builder.Services.AddRaccoonLandRabbitMqServiceEventConsumer(builder.Configuration);
```
