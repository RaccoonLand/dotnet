# 1. OutboxEventRecord & OutboxEventCategory

- **Package:** `RaccoonLand.Modules.Messaging.Abstractions`
- **Namespace:** `RaccoonLand.Modules.Messaging.Abstractions`

## OutboxEventCategory

| Constant | Value | Meaning |
| --- | --- | --- |
| `Domain` | `"Domain"` | In-service event; dispatched by `IDomainEventDispatcher`. |
| `Service` | `"Service"` | Integration event; published by `IServiceEventPublisher`. |

Helpers: `IsKnown(category)`, `EnsureKnown(category)` — unknown values are rejected.

These match the `Category` column written by `OutboxSaveChangesInterceptor`.

## OutboxEventRecord

Mirror of an outbox row used by the relay:

| Property | Source column | Notes |
| --- | --- | --- |
| `EventId` | `Id` | Same idempotency key as domain/service `EventId` (renamed for API consistency). |
| `Category` | `Category` | Must be known (`Domain` / `Service`). |
| `EventType` | `EventType` | Non-empty stable contract. |
| `AggregateType` | `AggregateType` | Non-empty. |
| `AggregateBusinessKey` | `AggregateBusinessKey` | |
| `Payload` | `Payload` | Non-empty JSON. |
| `CreatedBy` | `CreatedBy` | Optional. |
| `OccurredOnUtc` | `OccurredOnUtc` | |
| `CreatedOnUtc` | `CreatedOnUtc` | |
| `ClaimedOnUtc` | `ClaimedOnUtc` | Set by claim; used as fencing token. |

`ToClaim()` builds an `OutboxClaim(EventId, ClaimedOnUtc)` for `MarkProcessedAsync`.
