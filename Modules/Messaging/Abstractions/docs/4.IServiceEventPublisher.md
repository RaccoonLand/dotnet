# 4. IServiceEventPublisher

Port for publishing `Category = Service` outbox rows to an integration transport.

- **Package:** `RaccoonLand.Modules.Messaging.Abstractions`
- **Namespace:** `RaccoonLand.Modules.Messaging.Abstractions`

```csharp
public interface IServiceEventPublisher
{
    Task PublishAsync(OutboxEventRecord outboxEvent, CancellationToken cancellationToken = default);
}
```

## Notes

- Broker-specific packages (RabbitMQ, …) implement this interface.
- Register `AddRaccoonLandRabbitMqServiceEvents` and set `OutboxRelayOptions.ProcessServiceEvents = true`
  so the relay publishes Service rows.
- Success → relay marks processed; failure → retry after claim lease (at-least-once). Use `EventId` for consumer idempotency.

See also: [OutboxRelayOptions](5.Options.md), [RabbitMQ publisher](../RabbitMQ/docs/1.RabbitMqServiceEventPublisher.md), [Overview](0.Overview.md).
