# 2. S3Flavor

`S3Flavor` selects compatibility defaults:

| Flavor | Use case |
| --- | --- |
| `Aws` | Amazon S3 |
| `Minio` | MinIO and local S3-compatible stacks |
| `GenericS3Compatible` | Other S3-compatible providers |

Advanced overrides live under `Compatibility`:

```json
{
  "Compatibility": {
    "ForcePathStyle": true,
    "RequestTimeout": "00:05:00"
  }
}
```

Most deployments only need `Bucket`, `Region`, `Endpoint`, and `Flavor`.
