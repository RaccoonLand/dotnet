# 1. Registration

Register SQL Server FILESTREAM storage:

```csharp
services.AddRaccoonLandSqlServerFileStreamStorage(configuration);
```

## Surface

```csharp
public static IServiceCollection AddRaccoonLandSqlServerFileStreamStorage(
    this IServiceCollection services,
    IConfiguration configuration,
    string sectionName = SqlServerFileStreamStorageOptions.SectionName,
    string sharedSectionName = FileStorageOptions.SectionName);

public static IServiceCollection AddRaccoonLandSqlServerFileStreamStorage(
    this IServiceCollection services,
    IConfiguration configuration,
    Action<SqlServerFileStreamStorageOptions> configure,
    string sectionName = SqlServerFileStreamStorageOptions.SectionName,
    string sharedSectionName = FileStorageOptions.SectionName);
```

| Parameter | Default | Binds |
|-----------|---------|-------|
| `sectionName` | `FileStorage:SqlServer:FileStream` | `SqlServerFileStreamStorageOptions` |
| `sharedSectionName` | `FileStorage` | `FileStorageOptions` (shared guardrails) |

## Configuration

```json
{
  "FileStorage": {
    "MaxUploadBytes": 1073741824
  },
  "FileStorage:SqlServer:FileStream": {
    "ConnectionString": "Server=.;Database=AppDb;Trusted_Connection=True;TrustServerCertificate=True",
    "SchemaName": "dbo",
    "TableName": "FileBlobStreams",
    "FileGroupName": "FileStreamGroup",
    "EnsureSchema": false
  }
}
```
