# 1. Registration

Register SQL Server blob storage:

```csharp
services.AddRaccoonLandSqlServerFileStorage(configuration);
```

## Surface

```csharp
public static IServiceCollection AddRaccoonLandSqlServerFileStorage(
    this IServiceCollection services,
    IConfiguration configuration,
    string sectionName = SqlServerStorageOptions.SectionName,
    string sharedSectionName = FileStorageOptions.SectionName);

public static IServiceCollection AddRaccoonLandSqlServerFileStorage(
    this IServiceCollection services,
    IConfiguration configuration,
    Action<SqlServerStorageOptions> configure,
    string sectionName = SqlServerStorageOptions.SectionName,
    string sharedSectionName = FileStorageOptions.SectionName);
```

| Parameter | Default | Binds |
|-----------|---------|-------|
| `sectionName` | `FileStorage:SqlServer` | `SqlServerStorageOptions` |
| `sharedSectionName` | `FileStorage` | `FileStorageOptions` (shared guardrails) |

Custom section names:

```csharp
services.AddRaccoonLandSqlServerFileStorage(
    configuration,
    sectionName: "App:Uploads:SqlServer",
    sharedSectionName: "App:Uploads");
```

## Configuration

```json
{
  "FileStorage": {
    "MaxUploadBytes": 104857600,
    "AllowedContentTypes": [ "image/jpeg", "image/png", "application/pdf" ]
  },
  "FileStorage:SqlServer": {
    "ConnectionString": "Server=.;Database=AppDb;Trusted_Connection=True;TrustServerCertificate=True",
    "SchemaName": "dbo",
    "TableName": "FileBlobs",
    "EnsureSchema": true
  }
}
```
