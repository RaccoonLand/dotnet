# 12. FileUploadContent

Web-agnostic upload payload used by application and hosting adapters.

- **Package:** `RaccoonLand.Modules.FileStorage.Abstractions`
- **Namespace:** `RaccoonLand.Modules.FileStorage.Abstractions`
- **Source:** `src/Models/FileUploadContent.cs`

## Surface

```csharp
public sealed class FileUploadContent : IAsyncDisposable, IDisposable
{
    public required Stream Content { get; init; }
    public required string ContentType { get; init; }
    public long ContentLength { get; init; }
    public string? FileName { get; init; }
}
```

## Usage

Prefer building this from `IFormFile` via `RaccoonLand.Modules.FileStorage.AspNetCore`:

```csharp
await using var upload = file.ToFileUploadContent();
```

Then pass `upload.Content`, `upload.ContentType`, and `upload.ContentLength` into `FileStoragePutHelper.CreateRequest`.

See also: [AspNetCore registration](../AspNetCore/docs/1.Registration.md), [FilePutConstraints](11.FilePutConstraints.md).
