# 2. ExceptionHandlingOptions

`ExceptionHandlingOptions` holds developer-registered exception handlers that run **before** the built-in
[`DomainException`](../../../Core/Domain/docs/6.DomainException.md) handling in
[`ExceptionHandlingMiddleware`](1.ExceptionHandlingMiddleware.md).

- **Package:** `RaccoonLand.Modules.Middlewares.ExceptionHandlingMiddleware`
- **Namespace:** `RaccoonLand.Modules.Middlewares.ExceptionHandlingMiddleware`

## Surface

```csharp
public sealed class ExceptionHandlingOptions
{
    public IReadOnlyList<ExceptionHandlerRegistration> Handlers { get; }

    public ExceptionHandlingOptions On<TException>(
        Func<PipelineContext, TException, Task<bool>> handler)
        where TException : Exception;
}

public sealed record ExceptionHandlerRegistration(
    Type ExceptionType,
    Func<PipelineContext, Exception, Task<bool>> Handler);
```

`ExceptionHandlerRegistration` validates that `ExceptionType` and `Handler` are non-null and that
`ExceptionType` derives from `Exception`. Prefer `On<TException>`; constructing the record directly is
supported but fails fast on invalid arguments.

| Member | Description |
| --- | --- |
| `Handlers` | All registrations, in the order they were added. |
| `On<TException>(handler)` | Adds a handler for `TException` and derived types. Returns the options instance for chaining. |
| `ExceptionHandlerRegistration` | Stored registration shape (`ExceptionType` + `Handler`), validated at construction. |

## Handler contract

The handler receives the [`PipelineContext`](../../../Core/RequestProcessing/RequestProcessing.Abstractions/docs/3.PipelineContext.md)
and the caught exception. It must:

- Set `context.Response` when it handles the exception.
- Return `true` if the exception is fully handled (middleware stops and does not rethrow).
- Return `false` to defer to the next handler or built-in `DomainException` handling.

## Example

```csharp
builder.Services.AddRaccoonLandExceptionHandling(options =>
{
    options.On<ConcurrencyException>(async (context, exception) =>
    {
        context.Response = new PipelineResponse
        {
            Errors = [new PipelineMessage("CONCURRENCY_CONFLICT", "The resource was modified by another user.")],
        };
        return true;
    });
});
```

Handlers are checked **before** `DomainException`. Unhandled non-domain exceptions propagate to the caller.

See also: [ExceptionHandlingMiddleware](1.ExceptionHandlingMiddleware.md), [Registration](3.Registration.md).
