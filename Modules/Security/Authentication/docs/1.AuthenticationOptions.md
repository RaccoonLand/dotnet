# 1. AuthenticationOptions

`AuthenticationOptions` is the root configuration model bound from the `Authentication` section. It defines
default schemes, claim-mapping behaviour, and dictionaries of JWT Bearer and OpenID Connect schemes.

- **Package:** `RaccoonLand.Modules.Security.Authentication`
- **Namespace:** `RaccoonLand.Modules.Security.Authentication.Configuration`

## Surface

```csharp
public sealed class AuthenticationOptions
{
    public const string SectionName = "Authentication";

    public string? DefaultScheme { get; set; }
    public string? DefaultAuthenticateScheme { get; set; }
    public string? DefaultChallengeScheme { get; set; }
    public string? DefaultSignInScheme { get; set; }
    public string? DefaultSignOutScheme { get; set; }

    public bool DisableDefaultClaimMapping { get; set; } = true;

    public Dictionary<string, JwtBearerOptions> JwtBearer { get; set; }
    public Dictionary<string, OpenIdConnectOptions> OpenIdConnect { get; set; }
}
```

| Member | Description |
| --- | --- |
| `DefaultScheme` | Default authentication scheme for ASP.NET Core `AuthenticationOptions`. |
| `DefaultAuthenticateScheme` | Scheme used to populate `HttpContext.User`. |
| `DefaultChallengeScheme` | Scheme used when an endpoint requires authentication. |
| `DefaultSignInScheme` / `DefaultSignOutScheme` | Schemes for sign-in and sign-out operations. |
| `DisableDefaultClaimMapping` | When `true` (default), clears Microsoft's JWT claim type maps and sets `MapInboundClaims = false` on every registered scheme so claim types stay provider-native (`sub`, `email`, …). |
| `JwtBearer` | Named JWT Bearer schemes — see [SchemeOptions](2.SchemeOptions.md). |
| `OpenIdConnect` | Named OpenID Connect schemes — see [SchemeOptions](2.SchemeOptions.md). |

## Example

```json
{
  "Authentication": {
    "DefaultScheme": "Bearer",
    "DefaultAuthenticateScheme": "Bearer",
    "DefaultChallengeScheme": "Bearer",
    "DisableDefaultClaimMapping": true,
    "JwtBearer": {
      "Bearer": {
        "Authority": "https://login.example.com",
        "Audience": "my-api"
      }
    }
  }
}
```
