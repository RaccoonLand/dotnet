# 2. QueryAggregateRoot

`QueryAggregateRoot<TKey>` is the base class for **read models** — query-side projections of aggregates. Unlike
command-side aggregates, these are plain POCOs designed to be easy to query and shape.

- **Package:** `RaccoonLand.Modules.Persistence.SqlServer.Queries`
- **Namespace:** `RaccoonLand.Modules.Persistence.SqlServer.Queries`

## Surface

```csharp
public abstract class QueryAggregateRoot<TKey>
{
    public TKey Id { get; set; }
    public Guid BusinessKey { get; set; }
    public DateTimeOffset CreatedAtUtc { get; set; }
    public string? CreatedBy { get; set; }
    public DateTimeOffset? ModifiedAtUtc { get; set; }
    public string? ModifiedBy { get; set; }
    public Guid ConcurrencyToken { get; set; }
}
```

| Member | Description |
| --- | --- |
| `Id` | Primary key of the aggregate (type `TKey`). |
| `BusinessKey` | Stable business identifier of the aggregate. |
| `CreatedAtUtc` / `CreatedBy` | Creation audit stamps (mirrored from the command side). |
| `ModifiedAtUtc` / `ModifiedBy` | Last-modification audit stamps. |
| `ConcurrencyToken` | Optimistic-concurrency token — see below. |

## Design choices

- **Plain CLR types, no value objects.** Read models favor simplicity: scalar properties and navigation
  properties between aggregates are allowed, so projections and joins are straightforward.
- **No behavior / invariants.** These types carry data only; business rules live on the command side.
- **Concurrency token is always present.** Every read output exposes `ConcurrencyToken` so a caller can echo it
  back on the next command, enabling optimistic-concurrency checks end to end (read → edit → command).

## Usage

```csharp
public sealed class OrderReadModel : QueryAggregateRoot<long>
{
    public string CustomerName { get; set; } = string.Empty;
    public decimal Total { get; set; }
    public IReadOnlyList<OrderLineReadModel> Lines { get; set; } = [];
}
```

```csharp
var order = await db.Orders
    .Where(o => o.BusinessKey == businessKey)
    .Select(o => new OrderDto(o.Id, o.Total, o.ConcurrencyToken))
    .SingleOrDefaultAsync(cancellationToken);
```
